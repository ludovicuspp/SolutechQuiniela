# SolutechQuiniela

Quiniela interna para empleados de Solutech · FIFA World Cup 2026.

PWA construida con React 19 + Vite 8 + Supabase.

## Comandos

```bash
npm run dev    # Desarrollo
npm run build  # Build producción (PWA)
npm run lint   # ESLint
npm run test   # Tests
```

## Variables de entorno

```
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
VITE_FOOTBALL_API_KEY=
```
